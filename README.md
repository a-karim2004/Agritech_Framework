# Agritech App

Agritech App est un site statique d'exemple pour un projet agricole (météo, indicateurs FAO, formations). Ce dépôt sert de terrain d'apprentissage pour Bootstrap et la construction d'interfaces responsives.

**But**: te familiariser avec Bootstrap 5, organiser un petit site statique et appliquer de bonnes pratiques (accessibilité, performance, structure).

**Aperçu rapide**
- Page principale: `index.html`
- Pages secondaires: `formation.html`, `deco.html`, `Mdb.html`, `voir.html`
- Styles personnalisés: `css/style.css`
- Images: `image/`

## Prérequis

- Navigateur moderne (Chrome / Firefox / Safari)
- Editeur (VS Code recommandé)
- Terminal Zsh (macOS)
- Python 3 (optionnel pour servir localement)
